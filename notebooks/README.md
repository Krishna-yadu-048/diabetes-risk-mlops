# Notebooks

This folder is for exploratory data analysis (EDA) and feature engineering experiments.

Notebooks here are scratch space — once something is proven useful it gets moved into `src/`.

## Suggested notebooks

- `01_eda.ipynb` — Distribution of each feature, class balance, correlation heatmap
- `02_feature_engineering.ipynb` — Testing derived features before adding them to the pipeline
- `03_model_comparison.ipynb` — Quick sklearn comparisons before committing to train.py
